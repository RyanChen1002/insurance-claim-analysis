
# Insurance Claim Analysis

## Project Overview

This project investigates whether smokers tend to have higher insurance claim amounts than non-smokers.

## Research Question

Do smokers have higher insurance claims than non-smokers?

## Data Source

The dataset comes from Kaggle:

https://www.kaggle.com/datasets/thedevastator/insurance-claim-analysis-demographic-and-health

The raw CSV file is stored locally in the `data/` folder, but it is not committed to this GitHub repository.

## Unit of Analysis

Each row represents one patient or insurance record.

## Key Variables

- `claim`: insurance claim amount
- `smoker`: whether the person is a smoker
- `age`: age of the person
- `gender`: gender
- `bmi`: body mass index
- `bloodpressure`: blood pressure
- `diabetic`: whether the person is diabetic
- `children`: number of children
- `region`: region of the person

## Planned Analysis

This project will use:

- descriptive statistics
- data visualization
- a permutation test
- bootstrap confidence intervals

## Hypothesis

Null hypothesis: There is no difference in insurance claim amounts between smokers and non-smokers.

Alternative hypothesis: Smokers have higher insurance claim amounts than non-smokers.
